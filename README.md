# makrivich.github.io

Персональный веб-сайт и коллекция проектов.

## 📁 Структура проекта

| Директория | Описание |
|------------|----------|
| `dnevnik/` | Электронный дневник с данными классов 10A и 10B |
| `event/` | Модуль событий с интерактивным интерфейсом |
| `game/` | Игра (браузерная) |
| `project/` | Основной проект с генерацией ID кодов |
| `school/` | Учебные материалы (химия, эксперименты) |
| `taskmanager/` | Менеджер задач |

## 🚀 Запуск

Проект является статическим веб-сайтом. Для локального просмотра:

1. Откройте `index.html` любого из модулей в браузере
2. Или используйте локальный сервер:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Затем откройте http://localhost:8000
   ```
